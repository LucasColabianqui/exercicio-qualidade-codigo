# Exercício - Qualidade de Código

Aluno: Lucas Colabianqui

Linguagem utilizada: JavaScript

## Descrição

Este repositório apresenta exemplos comparativos entre código de má qualidade e código de boa qualidade.

Cada pasta demonstra um critério de qualidade de código como legibilidade, documentação, reutilização e segurança.

## Estrutura

Cada pasta contém:

- código ruim
- código melhorado
- explicação das melhorias

## Objetivo

Demonstrar boas práticas de desenvolvimento e qualidade de software.
