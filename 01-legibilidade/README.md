# Legibilidade

## Problema no código ruim
O código utiliza nomes de funções e variáveis pouco descritivos, como "c", "a" e "b".

## Melhorias no código bom
Foram utilizados nomes claros e significativos como "somar" e "numero1".

## Princípios aplicados
- Nomes significativos
- Código autoexplicativo
